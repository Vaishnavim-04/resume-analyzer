# AI Resume Analyzer

AI Resume Analyzer is a web-based application that evaluates resumes against job descriptions using rule-based logic. It calculates a match score, identifies skill gaps, and provides improvement suggestions.

## Features
- Match score calculation
- Skill gap detection
- Domain mismatch detection
- Suggestions for improvement
- PDF resume support
- Interactive UI

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express

## How to Run
1. Open terminal
2. Run: node server/server.js
3. Open: http://localhost:3000
